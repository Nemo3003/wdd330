An example of the touch event is touchend, which fires when the user stops touching the screen.


Example: mouseParagraph.addEventListener('touchend', ()=> console.log('Touch stopped'));


touchmove, touchenter, touchleave, touchcancel. I believe that most of them are self-explanatory


touchmove occurs when the user touches the screen and moves it without leaving. touchenter occurs when the user is touching the screen and passes over an element that had a touch event
touchleave occurs when the user is still touching the screen but leaves the specific event that was triggered
touchcancel occurs when a touch is interrupted, like too many fingers touching the screen


Swip events do not exist as "swipstart" for example, instead they are created convining touchstart, touchmove, and touchleave.
There are two libraries that made it possible to implement gestures, such are: hammer.js and zingtouch.
