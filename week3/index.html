Object Method: this

As I understood, the "this" object will point to the current object


This would allow us to call a certain property from the object. So, if the object is "Person" and you say "this.name"


You refer to the object "Person" and to its property "name" therefore if the name was "John Doe" you will have "John Doe" as a result.

It can also used in a function, so it is not necessary to have it inside a class


What is interesting to me is that arrow functions do not have "this".
