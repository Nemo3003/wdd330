The Browser Object Model - BOM

The Browser Object Model (or BOM for short) is a collection of properties and methods that contain information about the browser and computer screen
Properties and methods are available through the window object.

The BOM Only Makes Sense in a Browser Environment utilizing global variables
x = 6;
window.x; // 6
window.x === x; // true

Methods of the global object -> isNAN() and parseInt() - these are also methods of the Number object (Number.isNAN(4.2)// false)
window.parseInt(4.2); // 4
window.isNaN(4.2); // false
These can produce dialogs in the browsers: alert() , confirm() , and prompt()

Window properties

  • closed - Returns a boolean true if a window is closed.

  • console - Returns the Console Object for the window.

  • defaultStatus - Deprecated.

  • document - Returns the Document object for the window.

  • frameElement - Returns the frame in which the window runs.

  • frames - Returns all window objects running in the window.

  • history - Returns the History object for the window.

  • innerHeight - Returns the height of the window's content area (viewport) including scrollbars

  • innerWidth - Returns the width of a window's content area (viewport) including scrollbars

  • length - Returns the number of iframe elements in the current window

  • localStorage - Allows to save key/value pairs in a web browser. Stores the data with no expiration date

  • location - Returns the Location object for the window.

  • name - Sets or returns the name of a window

  • navigator - Returns the Navigator object for the window.

  • opener - Returns a reference to the window that created the window

  • outerHeight - Returns the height of the browser window, including toolbars/scrollbars

  • outerWidth - Returns the width of the browser window, including toolbars/scrollbars

  • pageXOffset - Returns the pixels the current document has been scrolled (horizontally) from the upper left corner of the window

  • pageYOffset - Returns the pixels the current document has been scrolled (vertically) from the upper left corner of the window

  • parent - Returns the parent window of the current window

  • screen - Returns the Screen object for the window

  • screenLeft - Returns the horizontal coordinate of the window relative to the screen

  • screenTop - Returns the vertical coordinate of the window relative to the screen

  • screenX - Returns the horizontal coordinate of the window relative to the screen

  • screenY - Returns the vertical coordinate of the window relative to the screen

  • sessionStorage - Allows to save key/value pairs in a web browser. Stores the data for one session

  • scrollX - An alias of pageXOffset

  • scrollY - An alias of pageYOffset

  • self - Returns the current window

  • status - Deprecated. Avoid using it.

  • top - Returns the topmost browser window


Window Object Methods

  • alert() - Displays an alert box with a message and an OK button

  • atob() - Decodes a base-64 encoded string

  • blur() - Removes focus from the current window

  • btoa() - Encodes a string in base-64

  • clearInterval() - Clears a timer set with setInterval()

  • clearTimeout() - Clears a timer set with setTimeout()

  • close() - Closes the current window

  • confirm() - Displays a dialog box with a message and an OK and a Cancel button

  • focus() - Sets focus to the current window

  • getComputedStyle() - Gets the current computed CSS styles applied to an element

  • getSelection() - Returns a Selection object representing the range of text selected by the user

  • matchMedia() - Returns a MediaQueryList object representing the specified CSS media query string

  • moveBy() - Moves a window relative to its current position

  • moveTo() - Moves a window to the specified position

  • open() - Opens a new browser window

  • print() - Prints the content of the current window

  • prompt() - Displays a dialog box that prompts the visitor for input

  • requestAnimationFrame() - Requests the browser to call a function to update an animation before the next repaint

  • resizeBy() - Resizes the window by the specified pixels

  • resizeTo() - Resizes the window to the specified width and height

  • scroll() - Deprecated. This method has been replaced by the scrollTo() method.

  • scrollBy() - Scrolls the document by the specified number of pixels

  • scrollTo() - Scrolls the document to the specified coordinates

  • setInterval() - Calls a function or evaluates an expression at specified intervals (in milliseconds)

  • setTimeout() - Calls a function or evaluates an expression after a specified number of milliseconds

  • stop() - Stops the window from loading

Cookies

  • document.cookie - Returns the value of the cookie with the specified name

  • document.cookie = "name=value; expires=date; path=path; domain=domain; secure" - Sets a cookie with the specified name and value

    • Animations

    • requestAnimationFrame() - Requests the browser to call a function to update an animation before the next repaint

      HTML5 APIs

      Data-

      Data - attribute allows you to embedd data in a web page using custom attributes that are ignored by the browser.
      These are used by a Javascript program.

      Must start with data-
      Only use lowercase letters, number, hyphens, dots, colons or underscores.

      Include an optional string value.

       data-powers = 'flight superSpeed'
      
    data-rating = '5'
      
    data-dropdown
      
    data-user = 'DAZ'
      
    data-max-length = '32'

      Drop the data- in javascript

      To access the attribute, powers is used as if it is a property of the dataset object. If a data- attribute's
      name contains hyphens, they are replaced with camel-case notation, so data-max-length would be accessed using
      dataset.maxLength

      Web storage API

      Works similar to cookies but has fewer restrictions and more storage.
      It allows you to store information about users and application information for later usage.
      The data stored does not automatically expire like it does with cookies.

      localStorage.variable = 'something'


      console.log(localStorage.variable)

      to remove simply use delete:
      Delete localStorage.variable and localStorage.clear() removes all local storage data

      Notification API

      It lets you have a pop up on user screen. It requires user permission before you can use it.
      use requestpermission() method to get the permission required.
      if(window.Notification) {
        Notification.requestPermission()
        .then((permission) => {
            if(Notification.permission === 'granted') {
            new Notification('Hello JavaScript!');
            }
        });
        }
      It can even be used to send the user to a new page if they click on the notification: 
      notification.addEventListener('click', () => {
            window.open('https://sitepoint.com')
            }, false);

      Multimedia

      Before HTML5 it was difficult to display videos or audio in the browser, but after HTML5 we can use the video and
      audio tag that allows us to display video or audio.

      • The play event, which fires when the clip starts and when it resumes after a pause.
      • The pause event, which fires when the clip is paused.
      • The volumechange event, which fires when the volume is changed.
      • The loadedmetadata event, which we saw in the note above, and which fires when all the video's metadata has loaded.

      video.addEventListener('pause', () => {
    console.log('The video has been paused'); }, false)
      There are lots of other APIs available for HTML5, but I have not covered them all.
